# Galaxycity

## Run bot

### Windows

#### Install packages:

```sh
pip install -r requirements.txt
```
or

```sh
python -m pip install -r requirements.txt
```
#### Run:

```sh
set TOKEN=bot_token
py -3 src/boy.py
```
### Linux

#### Install packages:

```bash
$ pip3 install -r requirements.txt
```
or

```bash
$ python3 -m pip install -r requirements.txt
```
#### Run:

```bash
$ export TOKEN=bot_token
$ python3 src/bot.py
```
